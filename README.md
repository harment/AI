# 🌿 المساعد الذّكاليّ في تعليم العربية لغير الناطقين بها

تطبيق ويب تعليمي تفاعلي مدعوم بالذكاء الاصطناعي والتلعيب التعليمي.

## ✨ المميزات

- 📱 تصميم متجاوب يعمل على الجوال وسطح المكتب
- 📲 PWA – إمكانية إضافة اختصار للشاشة الرئيسية
- 🎓 لوحة الطالب: مقررات، دروس، ألعاب تعليمية، نقاط، علماء نحو
- 👨‍🏫 لوحة الأستاذ: إدارة شاملة للطلاب، المقررات، الدروس، الأسئلة، التحليلات
- 🤖 ذكاء اصطناعي: توليد أسئلة، عروض تقديمية، معلومات علماء (Gemini / OpenAI)
- 🎮 ألعاب تعليمية: مغامرة الجبل ⛰️، المتاهة 🌀، البحر ⛵
- 📊 تحليلات وتصدير Excel
- 💬 شات بوت مساعد ذكي

## 🚀 التثبيت

### المتطلبات
- PHP 8.0+
- MySQL 5.7+ / MariaDB 10.4+
- Apache مع mod_rewrite

### الخطوات

1. استنساخ المستودع: `git clone https://github.com/harment/AI.git`
2. إنشاء ملف الإعدادات: `cp .env.example .env` ثم عدّل `.env` ببيانات قاعدة البيانات
3. إنشاء قاعدة البيانات: `mysql -u root -p < db/schema.sql`
4. فحص الإعداد: افتح `setup.php`

# ملفات الأصوات للعبة المغامرة

يجب إضافة الملفات الصوتية التالية:

- `correct.mp3` - صوت الإجابة الصحيحة
- `wrong.mp3` - صوت الإجابة الخاطئة
- `win.mp3` - صوت الفوز والوصول للقمة
- `lose.mp3` - صوت الخسارة والسقوط
- `progress.mp3` - صوت التقدم في الرحلة

يمكن استخدام مواقع مثل:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

للحصول على ملفات صوتية مجانية.
